### Viewing Modes

- [All Stains (Sequential)](https://images.patolojiatlasi.com/metaplasia/all.html) - Browse HE, trypsin stains in sequence
- [Side by Side Comparison](https://images.patolojiatlasi.com/metaplasia/sidebyside.html) - View stains in synchronized panels
- [Curtain Comparison](https://images.patolojiatlasi.com/metaplasia/curtain.html) - Overlay two stains with a draggable slider

### Görüntüleme Modları

- [Tüm Boyamalar (Sıralı)](https://images.patolojiatlasi.com/metaplasia/all.html) - Tüm boyamaları sırayla inceleyin
- [Yan Yana Karşılaştırma](https://images.patolojiatlasi.com/metaplasia/sidebyside.html) - Boyamaları eş zamanlı panellerde görüntüleyin
- [Perde Karşılaştırma](https://images.patolojiatlasi.com/metaplasia/curtain.html) - İki boyamayı sürgülü perde ile karşılaştırın
